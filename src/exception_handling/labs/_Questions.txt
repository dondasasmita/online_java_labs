1.) What is exception handling?
It is a method that handles a problem that might arise during the execution of a program.

2.) What is the exception hierarchy? What is the difference between the Exception class and the Error class?
Exception hierarchy is an organization of Java exception classes.
Exception class is a class that can be handled.
Error class is an a class that cannot be handled.

3.) What is a try/catch?
These are blocks of code that contain set of statements where an exception can occur. A try block is
always followed by a catch block, which handles the exception that occurs associated try block.
